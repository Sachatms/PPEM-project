This file aims to give you a complete way to build and launch the hello world program on a C6678 target

1. Clone the repository `Sachatms/PPEM-project` from GitHub
2. Launch Code Composer Studio v6.1.3
3. set the Workspace directory at <path\_to\_the\_cloned\_repository>/PPEM-project/C6678/
4. Then follow in order, step 20, 22, 23, 24, 25 and 26 of the second lab assignement and you're done
